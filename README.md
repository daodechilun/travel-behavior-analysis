# 旅游用户行为分析与消费预测系统

基于 **62,200 条**高德地图旅游用户行为数据，完成从数据清洗到机器学习建模的全链路分析。建立 **14 个预测模型**对比，客流预测达 **R²=0.750**、消费预测达 **F1=0.891**。

## 核心成果

- **建模精度**：客流预测 R²=0.750，消费预测 F1=0.891（14 个模型对比后的最优结果）
- **特征洞察**：通过消融发现用户画像是消费预测的关键因子——移除后 R² 从 0.75 骤降至 0.16，量化证明了画像价值
- **业务结论**：高价值用户占 21.4% 却贡献大部分消费；核心用户消费水平是普通用户 2.3 倍；预订转化率 24.8%

## 数据概况

- 数据量：62,200 条用户行为记录
- 覆盖：8 个城市、4 个品类（美食/酒店/景点/旅游服务）
- 时间跨度：7 天
- 行为类型：浏览 / 收藏 / 预订 / 评价

## 技术实现（10 个分析模块）

`main.py` 单文件串联全流程，一键复现：

| 阶段 | 模块 | 方法 |
|------|------|------|
| 数据处理 | 清洗 + EDA | 缺失值/异常值/去重/日期解析 |
| 行为分析 | 路径分析、消费偏好、时空特征、转化漏斗 | 漏斗模型、四维加权评分推荐 |
| 用户建模 | 画像与分群 | **RFM + K-Means** |
| 关联挖掘 | 频繁项集 | **Apriori**（mlxtend） |
| 预测建模 | 客流预测、消费预测 | 14 模型对比，最优 R²=0.75 / F1=0.891 |
| 可视化 | 城市雷达、消费热力图 | matplotlib + seaborn |

## 运行

```bash
pip install pandas numpy scikit-learn matplotlib seaborn mlxtend python-docx
python main.py
```

运行后自动完成：数据清洗 → 10 个分析模块 → 8 张可视化图表输出到 `output_figures/`。
